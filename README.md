# html-and-css

## Bootstrap Movie Website
A responsive Bootstrap project showcasing movies, showtimes, and deals. Features:
- Navbar with bookmarks
- Jumbotron
- Movie cards with images, titles, descriptions
- Pill badges for showtimes

## Portfolio Website
A colorful HTML & CSS portfolio for Wael Khlaif. Features:
- Fixed navbar
- Centered image with filters and rounded edges
- Styled table
- Text overlays on images and videos
- Two-column layout
- Hover effects
- Image overlay fade
- Autoplay looping video
- Styled footer
